## Set-up

There are different steps to setup it, to avoid complexity the shell script are weapon to use
on you need to do this

#### <span style="color: lightblue;">Check Your system has any C++ complier using below command.</span>

```bash
  g++ --version # or clang --version
```

if not then install it. and then go further.

#### <span style="color: lightblue;">Check CMake</span>.

```bash
  cmake --version
```
if not then install it. and then go further.

#### <span style="color: lightblue">Build Your project.</span>

```bash

  chmod +x ./build.sh &&  ./build.sh 

  # for building project
```

#### <span style="color: lightblue">Runing the Project.</span>

<span style="color: red;">* before run you should add your target files </span>

```bash   

  chmod +x ./start.sh && ./start.sh 
  
  # for run the project
```
