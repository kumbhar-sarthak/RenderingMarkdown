# Rendring Markdown

C++ project inspired by Common Mark, To handle multiple files, a thread based file processing that helps to parse larger files.

## Set-up

There are different steps to setup it, to avoid complexity the shell script are weapon to use
on you need to do this

```bash
  ./build-temp.sh # for building project

  ./temp.sh # for run the project
```
<span style="color:red">As of now you manually add files path in ./temp.sh in working mode to change this sorry.</span>
